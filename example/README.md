# Example Microscope Game

This folder contains an example Microscope Canvas game that demonstrates all the features of the application.

## The Rise and Fall of the Crystal Empire

**File:** `crystal-empire.json`

This example showcases:

### Big Picture
- **Premise**: The overarching concept for the history
- **Bookends**: Starting and ending points that frame the timeline

### Palette
- **Yes List**: Elements explicitly allowed in the world (magic crystals, political intrigue, prophecies, etc.)
- **No List**: Elements explicitly banned (aliens, time travel, resurrection, etc.)

### Focuses
- Multiple focuses that guide exploration, including:
  - The Nature of Power
  - The Crystal Singers
  - What Was Lost

### Players
- Single player setup (typical for solo Microscope games)

### Legacies
- Recurring elements that persist through history:
  - The Heartstone (an artifact)
  - House Luminari (a dynasty)
  - The Silent Order (an organization)

### Timeline Structure
The game includes 4 periods with varying tones:

1. **The Age of Discovery** (Light) - Beginning of the crystal civilization
2. **The Golden Empire** (Light) - Height of crystal power
3. **The War of Shadows** (Dark) - Conflict and collapse
4. **The Long Twilight** (Dark) - Decline and ending

### Events and Scenes
Each period contains multiple events, and several events have detailed scenes that explore:
- **Questions**: What the scene aims to answer
- **Answers**: What was discovered through play
- **Descriptions**: Narrative details

## How to Use

1. Open Microscope Canvas
2. Click "Import Game"
3. Select `crystal-empire.json`
4. Explore the timeline to see how periods, events, and scenes are structured

## Creating Your Own Games

Use this example as inspiration! The best Microscope games:
- Have a clear premise that spans a significant period of time
- Include both light and dark elements for contrast
- Build on focuses and legacies to create thematic cohesion
- Use scenes to explore pivotal moments in detail
